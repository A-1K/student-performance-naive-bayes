Student Performance Prediction using Mixed Naive Bayes

A Bayesian Network model predicting student academic performance tiers — High, Medium, Low — using a custom mixed-likelihood Naive Bayes built entirely from scratch and benchmarked against scikit-learn.

86.01% Accuracy 2,392 Student Records Kaggle Dataset 3-Class Classification
01 · Overview

What was built

Custom Implementation

A Mixed Naive Bayes classifier built from scratch with Gaussian likelihoods for continuous features, Bernoulli likelihoods for binary features, and Laplace-smoothed categorical likelihoods for multi-class features. Uses log-probabilities to prevent numerical underflow.

Baseline Comparison

A scikit-learn GaussianNB model trained on the same data and split for direct comparison. The custom implementation treats feature types individually rather than forcing all features through a single Gaussian assumption.

Dataset

Kaggle — rabieelkharoua/students-performance-dataset
2,392 records · 9 features · 3-class target derived from GradeClass.

Target Classes

High — GradeClass 0 or 1
Medium — GradeClass 2
Low — GradeClass 3 or 4

03 · Methodology

Probabilistic Model

P(Y | X₁…X₉) ∝ P(Y) × Π P(Xᵢ | Y) Log-posterior = log P(Y) + Σ log P(Xᵢ | Y) Predicted class = argmax [ log-posterior ]
  • Conditional Independence: All features assumed independent given the class label — the core Naive Bayes assumption.
  • Mixed Likelihoods: Each feature type uses the statistically appropriate distribution rather than a single Gaussian for everything.
  • Laplace Smoothing: Applied to categorical features to handle zero-probability edge cases in unseen category values.
  • Log-probabilities: Products of small probabilities replaced with sums of logs to avoid floating-point underflow.
  • 80/20 Split: Fixed random seed 42 for reproducibility across all experiments.

06 · Key Observations

What the model tells us

  • Custom outperforms baseline: Mixed likelihoods (86.01%) beat the all-Gaussian GaussianNB (85.59%) by treating each feature type correctly.
  • Top predictors: GPA, Absences, and StudyTimeWeekly carry the most discriminative signal.
  • Medium class difficulty: Precision for Medium is only 0.61 — it overlaps significantly with both High and Low groups in feature space.
  • Independence violations: GPA and Absences are correlated, breaking the Naive Bayes assumption — yet accuracy remains strong, confirming NB's robustness to mild violations.
